# E-commerce Books Store

This is a simple Headphones Store written in Java


# Tools used
- Java
- Android Studio IDE
- Android version 4.0 or later
- Android SDK 24-33


## Lessons Learned

In this project I learned how to use:
- Fragments
- Bottom Navigation
- RecylerView
- Drawer Navigation
